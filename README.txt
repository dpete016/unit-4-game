Crystal Collector
git@github.com:dpete016/unit-4-game.git

A web browser game that updates the HTML using the Jquery library

What do you do?
You have four crystals and a random number. You win the game by matching your total score to the random number. 
Clicking on the crystals adds up a specific number of points towards the total score. 
Value of each crystal are a mystery to solve and going over the total point results in a loss. 
The value of the crystal also changes each reset, so watch out for that and good luck.
